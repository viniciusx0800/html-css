# COMANDOS GIT

## ADICIONAR
1. GIT ADD . // PARA TODOS ARQUIVOS ALTERADOS
2. GIT ADD NOME-DO-ARQUIVO // ADICIONAR APENAS ARQUIVOS DIGITADOS

## COMMITAR
1. GIT COMMIT -M "MENASGEM"

## ENVIAR
1. GIT PUSH 
2. GIT PUSH -U ORIGIN MAIN

## PUXAR
1. GIT PULL
2. GIT FETCH

## STATUS LOCAL
1. GIT STATUS

## HISTORICO
1. GIT LOG
